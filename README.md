## 🛠 配置亮点

所有应用程序均已配置为：

* 自动将二进制文件添加到`PATH`
* 将持久化数据存储在`scoop/persist`目录中
* 在安装后应用常见配置（如注册表设置、配置文件等）

## 🧪 使用说明

将此仓库添加到Scoop：

```powershell
scoop bucket add apps https://github.com/morning-start/apps-bucket.git
scoop bucket add apps https://gitee.com/morning-start/apps-bucket.git
```
